#  CodSoft Data Science Internship

##  Overview
This repository contains the projects completed during my **Data Science Internship at CodSoft**.  
The internship focused on applying machine learning techniques using Python to solve real-world problems.

During this internship, I worked on data preprocessing, feature engineering, model building, and evaluation for both classification and regression tasks.

---

##  Projects Completed

### Iris Flower Classification
A classification model to predict the species of Iris flowers based on sepal and petal measurements.

- Type: Classification  
- Algorithms Used: Logistic Regression, Random Forest Classifier 
- Key Skills: Data preprocessing, feature encoding, model evaluation  

---

### Fraud Detection System
A machine learning model to detect fraudulent financial transactions.

- Type: Classification  
- Algorithms Used: Logistic Regression  
- Key Skills: Handling imbalanced datasets, precision and recall analysis  

---

### Sales Prediction System
A regression model to predict future sales based on historical data.

- Type: Regression
- Algorithm used: Linear Regression 
- Key Skills: Feature engineering, predictive modeling  

---

### Titanic Survival Prediction
A classification model to predict whether a passenger survived the Titanic disaster.

- Type: Classification  
- Algorithms Used: Logistic Regression
- Key Skills: Handling missing values, feature engineering, encoding categorical data

###  Movie Rating Prediction
A  Regression model to predict whether a passenger survived the Titanic disaster.

- Type: Regression
- Algorithms Used: linear Regression
- Key Skills: Handling missing values, feature engineering, encoding categorical data
